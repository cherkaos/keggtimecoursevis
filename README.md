# keggtimecoursevis
A visualization suite for time course information of a metabolic network. Generates an interactive, dsj3-based website.

## Introduction
This github is used to (1) generate a JSON network file from a KEGG metabolic network that can be used in plt or dsj3 for network visualization. The github also has code to (2) create a dsj3-based interface to look at dynamic metabolic data.

## Elements

The github is organized as follows:
* index.php - a PHP file that generates the network layout from the network JSON,the time course data, the annotation file
